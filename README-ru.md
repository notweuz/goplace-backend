# GoPlace

## Описание

Попытка создать платформу для пиксель-баттла по типу r/place.  
Это был мой первый «серьёзный» проект на Go.

## Состояние проекта

Проект сильно устарел. Я не думаю, что он будет нормально работать в текущем виде, так как писал его в первые месяцы изучения Go.

Большинство функций и логики нуждаются в оптимизации или полной переработке.

При этом я, скорее всего, оставлю текущий подход с отдачей пикселей в виде JSON, вместо генерации изображений по чанкам (как это делают почти все остальные пиксель-баттлы).

## Планы на проект

Я планирую полностью переписать проект, как только у меня появится достаточно свободного времени.  
Скорее всего, это произойдёт через 6–7 месяцев.
