## [Открыть README на русском](README-ru.md)

## Description

An attempt to create a pixel battle platform similar to r/place.  
This was my first "serious" Go project.

## State of the project

The project is heavily outdated. I don’t think it will work properly in its current state, as I wrote it during my very first months of learning Go.

Many features and parts of the logic need to be optimized or completely rewritten.

That said, I will most likely keep the current approach of serving pixels as JSON instead of generating chunk images (as is done in most other pixel battle platforms).

## Plans for the project

I plan to completely rewrite the project once I have enough free time.  
This will likely happen in 6–7 months.
